# 💬 Real-Time Chat App

A sleek and responsive real-time chat application built with **React**, **TypeScript**, **WebSockets**, and **Tailwind CSS**. 
Supports live messaging between users in the same room.

## 🚀 Features

- Real-time messaging with WebSocket
- Multiple users per room 
- Unique room codes
- Clean, modern UI

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Real-time Communication:** WebSocket (`ws` Node.js library)
- **State & Logic:** React Hooks (useEffect, useRef)
